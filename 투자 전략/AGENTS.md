## 1. 크립토 고유 트랙 조건

common-rules.md 트랙 판단에 추가:
- 트랙 A 추가 조건: 교차 참조 영향이 없거나 1~2곳
- 트랙 B 추가 조건: 교차 참조 영향 3곳 이상, 기존 분석/판단과 모순되는 내용, 새 섹션 추가

---

## 2. 절대 하지 않는 것

- 기존 분석의 결론을 자의적으로 변경하지 않는다. 수치 업데이트는 하되 판단은 사용자 지시만.
- 섹션 번호를 임의로 재배정하지 않는다.
- 부록(A~D) 순서를 변경하지 않는다.
- 변경 이력(부록 D) 기존 항목을 수정하지 않는다. 추가만.

## 3. 문서 서식 규칙

| 요소 | 규칙 |
|---|---|
| 섹션 제목 | ## Part, ### 소제목, #### 하위 |
| 표 | 파이프 구분. 수정 시 표 전체 교체 |
| 인용구 | > 블록. 핵심 통찰/원칙/경고 |
| 굵은 이탤릭 | ***텍스트***. 최종 요약/판정만 |
| 코드 | 사용하지 않음 |

변경 이력(부록 D): `- M/DD: [변경 요약]. [영향 섹션].`
